# FMP MCP Server銘柄分析プロンプト

## 基本プロンプト

```
[銘柄シンボル（例：AAPL）]について、FMP MCP Serverを使用して包括的なファンダメンタルズとテクニカル分析を実施し、視覚的なインフォグラフィックにまとめてください。

以下の分析を含めてください：

### ファンダメンタルズ分析
1. **企業概要**
   - 会社プロフィール（業界、市場、事業概要）
   - 時価総額、従業員数、設立年

2. **財務指標**
   - 最新の損益計算書データ（売上、純利益、EPS）
   - 主要な財務比率（PER、PBR、ROE、ROA）
   - 債務比率と流動性指標

3. **成長性分析**
   - 売上成長率（YoY、QoQ）
   - 利益成長率
   - 将来の成長見通し

4. **配当情報**
   - 配当利回り
   - 配当履歴
   - 配当性向

### テクニカル分析
1. **価格動向**
   - 現在の株価と変動率
   - 52週高値・安値との比較
   - 主要な移動平均線との位置関係

2. **テクニカル指標**
   - EMA（指数移動平均）
   - 出来高分析
   - 相対的強弱指数（RSI）等

3. **市場データ**
   - 最新の取引データ
   - 時間外取引情報
   - 市場でのポジション

### アナリスト評価
1. **投資判定**
   - アナリストレーティング
   - 目標株価
   - 最新の価格目標ニュース

2. **財務予想**
   - EPS予想
   - 売上予想
   - 成長率予想

### 総合評価とリスク分析
- 投資魅力度の総合スコア
- 主要なリスク要因
- 業界内での競争ポジション

最終的に、これらの分析結果を以下の要素を含む視覚的なインフォグラフィックにまとめてください：
- カラフルなチャートとグラフ
- 重要な数値のハイライト表示
- 直感的に理解できるレイアウト
- 投資判断に役立つサマリー
- レスポンシブデザイン
- 言語は英語
```

## カスタマイズオプション

### 特定の分析に焦点を当てたい場合：
```
[銘柄シンボル]について、特に[配当分析/成長性/バリュエーション/テクニカル]に重点を置いた分析を行い、インフォグラフィックにまとめてください。
```

### 比較分析を含める場合：
```
[銘柄シンボル1]と[銘柄シンボル2]を比較分析し、両社のファンダメンタルズとテクニカル指標を対比したインフォグラフィックを作成してください。
```

### 特定の期間での分析：
```
[銘柄シンボル]について、過去[期間]の業績とテクニカル動向を分析し、将来の見通しを含めたインフォグラフィックを作成してください。
```

## 期待される出力

1. **データ取得フェーズ**
   - 複数のFMP API呼び出しによる包括的なデータ収集
   - 財務データ、市場データ、アナリスト予想の統合

2. **分析フェーズ**
   - 定量的・定性的な分析
   - 業界標準との比較
   - リスク・リターン評価

3. **視覚化フェーズ**
   - HTMLアーティファクトによるインタラクティブなインフォグラフィック
   - チャート、グラフ、メトリクス表示
   - モバイルフレンドリーなデザイン

## 注意事項

- 分析は投資アドバイスではなく、情報提供目的であることを明記
- データの取得日時を表示
- 市場の変動性に関する免責事項を含める
- 最新の市場状況を反映したリアルタイムデータを使用
